# Marketing-ROI-Analysis
End-to-end marketing analytics project analyzing 7,500+ campaigns to optimize $5.2M ad spend using Python, AWS, Power BI, and Auto ML

 📊 Marketing ROI Dashboard - Power BI Project

 📋 Project Overview
A comprehensive 3-page interactive dashboard analyzing 7,500+ marketing campaigns to optimize ROI and reduce ad spend waste.

## 🎯 Business Objectives
- Identify channels with highest ROI
- Find audience segments that convert best
- Detect wasted spend on low-ROI campaigns
- Provide actionable recommendations for budget optimization

📈 Key Findings
- YouTube delivers 7.2x ROI (best channel)
- **Women 35-44 convert 35% better than average
- 30-day campaigns outperform 60-day by 23%
- $420K identified in wasted spend on low-ROI campaigns
- $1.17M total optimization potential
