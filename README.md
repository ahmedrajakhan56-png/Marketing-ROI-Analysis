## 🚀 Quick Start Guide

```bash
# 1. Clone the Google-style repository
git clone https://github.com/ahmedrajakhan56-png/Marketing-ROI-Analysis.git
cd Marketing-ROI-Analysis

# 2. Install dependencies
pip install -r requirements.txt

# 3. Configure AWS (for cloud analytics)
aws configure
# Enter: AWS Access Key, Secret Key, region (us-east-1)

# 4. Run ETL pipeline
python src/etl_pipeline.py

# 5. Train Auto ML model
python src/train_auto_ml.py

# 6. Launch Power BI dashboard
# Open powerbi/marketing_roi_dashboard.pbix
```
